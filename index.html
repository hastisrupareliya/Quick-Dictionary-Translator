Quick Dictionary + Translator

Quick Dictionary + Translator is a Chrome extension that lets you instantly see word definitions, example sentences, and translations while browsing the web.

Features

  • Highlight any word to see definitions instantly
  • Multiple definitions and example sentences
  • Translate into multiple languages (Spanish, French, German, Hindi, etc.)
  • Toggle example sentences on/off
  • Copy translation or hear pronunciation with one click
  • Modern animated popup with smooth UI
  • Remembers your preferred translation language

How to Use

1. Install the extension from Chrome Web Store.
2. Highlight any word on a webpage.
3. A popup will appear showing the definition, examples, and translation.
4. Use the language dropdown to change translation language.
